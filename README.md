# filtros-distribucion-cordoba

## Propósito del proyecto
Este repositorio organiza información clave para un proyecto de distribución de filtros en Córdoba. Permite relevar clientes potenciales (lubricentros), mapear equivalencias entre marcas de filtros y analizar márgenes frente a la competencia.

## Uso de cada planilla
- `data/relevamiento_lubricentros.csv`: registra datos de contacto, hábitos de compra y observaciones de cada lubricentro. Se usa como base de prospección y seguimiento comercial.
- `data/equivalencias_millard.csv`: almacena equivalencias entre códigos Millard y otras marcas por tipo de filtro, vehículo y motor. Sirve para responder consultas de compatibilidad.
- `data/comparador_precios.csv`: compara costos, precios de venta y márgenes frente a competidores. Ayuda a definir estrategia de precios.

## Flujo recomendado
1. Relevar lubricentros y completar `relevamiento_lubricentros.csv`.
2. Cargar equivalencias en `equivalencias_millard.csv`.
3. Solicitar precios de mercado y completar `comparador_precios.csv`.
4. Analizar márgenes para ajustar oferta y condiciones comerciales.
